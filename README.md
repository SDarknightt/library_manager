# library_manager

Bem-vindo ao Sistema de Biblioteca! Este é um projeto que permite gerenciar uma biblioteca, utilizando o PostgreSQL como banco de dados, Java para a programação do back-end, e HTML e CSS para o front-end, sem o uso de frameworks ou bibliotecas adicionais.

## Funcionalidades Principais

- Autenticação de Usuário: O sistema permite criar contas de usuário com login e senha para acessar as funcionalidades do sistema.

- Gerenciamento de Clientes: É possível cadastrar e visualizar informações dos clientes.
- Gerenciamento de Livros: O sistema permite cadastrar e visualizar as informações dos livros da biblioteca, incluindo nome e gênero.
- Reservas de Livros: Os usuários podem fazer reservas de livros disponíveis na biblioteca, mantendo um controle das reservas ativas.

## Tecnologias Utilizadas
- Linguagem de Programação: Java
- Banco de Dados: PostgreSQL
- Front-end: HTML e CSS
